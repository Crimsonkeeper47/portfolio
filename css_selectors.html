Selector Example Description
Class Selector .classname Selects elements with a specific class name.
Multi-class Selector .example1.name4 Selects all elements with both example1 and name4 set within its class attribute
Descendant-Class Selector .example3 .class5 Selects all elements with class5 within the ancestor selector example3.
ID Selector #idname Selects the element with the id name.
Asterisk(*) * Selects all elements.
Element Selector p Selects all p elements.
Element-Class Selector p.classname Selects all p elements with classname.
Element-div Selector div,h1 Selects all div elements and all h1 elements.
Element-Internal (Descendant) Selector div h1 Selects all div elements inside h1 elements.
Element-External (Ancestor) Selector div > h1 Selects all h1 elements inside div elements.
Direct Element Selector div + h1 Selects the first h1 element that is placed immediately after div elements.
Adjacent Element Selector p ~ h1 Selects every p element that is proceeded by a h1 element.
Attribute Selector [attribute] Selects the target attribute.
Attribute-Value Selector [attribute=value] Selects all elements with the target attribute set to the target value. (e.x target="_blank")
Attribute-Title Selector title~="_brick" Selects all elements with the title attribute containing the word brick.
Attribute-Language Selector lang|="en" Selects all elements with the lang attribute set to en or starting with "en-"
Attribute-Webpage Selector [href^="https"] Selects all (a) elements with the href attribute that starts with https.
Attribute-Image Selector a[href$=".pdf"] Selects all (a) elements with the href attribute that ends with .pdf.
Attribute-Website Selector a[href*="w3schools"] Selects all (a) elements with the href attribute that contains the substring "w3schools".
Active Web Link Selector a:active Selects the active link.
After-Element Selector p:after Inputs something after the content of each p element.
Before-Element Selector h1:before Inputs something before the content of each h1 element.
Checked Element Selector input:checked Selects all checked input elements.
Default-Element Selector input:default Selects the default input elements.
Disabled Element Selector input:disabled Selects all disabled input elements.
Empty Element Selector input:empty Selects every input element that has no children (including text nodes).
Enabled Element Selector input:enabled Selects all enabled input elements.
First-Child Element Selector p:first-child Selects every first child of the p element.
First-Letter Element Selector p:first-letter Selects the first letter of every p element.
First-Line Element Selector p:first-line Selects the first line of every p element.
First-of-Type Selector p:first-of-type Selects every p element that is the first p element of its parent.
Focus-Element Selector input:focus Selects the input element that has focus.
Full-Screen Selector html:fullscreen Selects the html element that is in full screen.
Hover Selector a:hover Selects the a element when the mouse is hovering over it.
Range Selector input:in-range Selects the input element that is within the specified range.
Indeterminate-Element Selector input:indeterminate Selects input element that are in an indeterminate state.
Invalid Element Selector input:invalid Selects the input element that has an invalid value.
Language/Font-Element Selector p:lang(it) Selects every p element with a lang attribute to it (Italian).
Last-Child Element Selector p:last-child Selects every last child of the p element.
Last-of-Type Selector p:last-of-type Selects every p element that is the last p element of its parent.
Link Element Selector a:link Selects the a element that has not been visited.
Marker Selector :marker Selects the markers of list items.
Not Element Selector input:not Selects every element that is not an input element.
Nth-Child Selector p:nth-child(2n) Selects every p element that is the second child of its parent.
Nth-Last-Child Selector p:nth-last-child(2n) Selects every p element that is the second child of its parent, counting from the last child.
Nth-Last-Of-Type Selector p:nth-last-of-type(2n) Selects every p element that is an even number, counting from the last of type.
Nth-Of-Type Selector p:nth-of-type(2n) Selects every p element that is the second p element to the parent.
Only-of-Type Selectors p:only-of-type Selects every p element that is the only p element of its parent.
Only-Child Selector p:only-child Selects every p element that is the only child of its parent.
Optional Element Selector input:optional Selects the input element that has an optional value (No specified or required attribute).
Outside-Range Element Selector input:out-of-range Selects the input element that is outside the specified range.
Placeholder Element Selector input:placeholder-shown Selects the input element that has a placeholder attribute.
Read-Only Element Selector input:read-only Selects the input element that has a read-only attribute specified.
Read-Write Element Selector input:read-write Selects the input element that has a read-write attribute NOT specified.
Required Element Selector input:required Selects the input element that has a required attribute specified.
Root Element Selector :root Selects the root element of the document.
Element Selection ::selection Selects the element that is selected by the user.
Target-Element Selector #news:target Selects the current #news element (clicked on a URL containing that anchor name).
Valid-Element Selector input:valid Selects the input element that has a valid value.
Visited-Site Element Selector a:visited Selects the a element that has been visited.
