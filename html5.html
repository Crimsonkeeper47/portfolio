Tags Description
!DOCTYPE html This tag is required to tell the browser that the document is an HTML5 document, this is a basic necessity to all HTML webpages.
HTML This tag is used to define the root of an HTML document.
Head This tag is used to define the header of an HTML document.
Title This tag is used to make a title for the HTML document.
Body This tag is used to define the body of an HTML document.
h1-h6 These tags are used to make large to small headings for an HTML document.
p This tag is used to make a paragraph for an HTML document.
a This tag is used to make a link for an HTML document.
img This tag is used to make an image for an HTML document.
ul This tag is used to make an unordered list for an HTML document.
ol This tag is used to make an ordered list for an HTML document.
li This tag is used to make a list item for an HTML document.
br This tag is used to make a line break for an HTML document.
hr This tag is used to make a horizontal line for an HTML document.
table This tag is used to make a table for an HTML document.
tr This tag is used to make a table row for an HTML document.
th This tag is used to make a table header for an HTML document.
td This tag is used to make a table data for an HTML document.
div This tag is used to make a division for an HTML document.
span This tag is used to make a span for an HTML document. This is commonly used in table cells or headers to span through a colum or a row.
header This tag is used to make a header for an HTML document.
footer This tag is used to make a footer for an HTML document.
main This tag is used to make a main for an HTML document.
section This tag is used to make a section for an HTML document.
nav This tag is used to make a navigation for an HTML document.
article This tag is used to make an article for an HTML document.
Advanced Tags Description
Comment (#) This tag is used to make a comment for an HTML document.
b This tag is used to make bold text for an HTML document.
